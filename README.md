## Project Overview
I built the "Flavors Kitchen" website as a multi-page project for my Web Development assignment. The objective was to design a functional, responsive, and visually appealing recipe site using only **HTML and CSS**. In compliance with the assignment rules, no JavaScript was utilized for any part of the website.

The site consists of a homepage and five detailed recipe pages, featuring a clean navigation system and a layout optimized for various screen sizes.

## Project Features
* **Semantic HTML Structure:** I employed semantic HTML tags to ensure the code is organized, accessible, and easy to read.
* **Recipe Database:** The site includes five distinct recipes: Butter Chicken, Margherita Pizza, Paneer Tikka Masala, Chocolate Brownies, and Caesar Salad. Each page provides specific ingredients, preparation times, difficulty levels, and step-by-step cooking instructions.
* **CSS Layouts:**
    * **Flexbox:** Used for the header and navigation bar to maintain alignment and spacing.
    * **CSS Grid:** Implemented on the homepage to create a structured, card-based layout for featured recipes.
    * **Box Model:** Careful application of margins, padding, and borders ensures the content is well-spaced and readable.
* **Responsive Design:** I integrated media queries in the CSS to ensure the website scales properly across mobile, tablet, and desktop devices. On smaller screens, the navigation links and recipe grid adjust to a vertical stack for better usability.
* **User Interface:** Interactive hover effects were added to links, buttons, and category pills to provide a dynamic feel without the use of scripts.
